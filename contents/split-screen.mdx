## HTML

```html
<div class="split-screen">
    <!-- Left content -->
    <div class="split-screen__half">...</div>

    <!-- Right content -->
    <div class="split-screen__half">...</div>
</div>
```

## CSS

```css
.split-screen {
    display: flex;
}

.split-screen__half {
    flex: 1;
}
```
