## HTML

```html index.html
<div class="sticky-sections">
    <section class="sticky-sections__section">...</section>

    <!-- Repeat other sections -->
    ...
</div>
```

## CSS

```css styles.css
.sticky-sections {
    height: 100%;
    overflow: scroll;
}

.sticky-sections__section {
    /* Take full size */
    height: 100%;
    width: 100%;

    /* Stick to the top */
    position: sticky;
    top: 0;
}
```
