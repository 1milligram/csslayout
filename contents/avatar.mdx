## HTML

```html index.html
<div class="avatar">
    <!-- Avatar image -->
    <img class="avatar__image" src="..." />
</div>
```

## CSS

```css styles.css
.avatar {
    /* Rounded border */
    border-radius: 50%;

    /* Center the content */
    align-items: center;
    display: flex;
    justify-content: center;

    /* Size */
    height: 4rem;
    width: 4rem;
}

.avatar__image {
    /* Size */
    height: 50%;
    width: 50%;
}
```
