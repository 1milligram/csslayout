## HTML

```html index.html
<div class="card">
    <!-- Cover -->
    <div class="card__cover">...</div>

    <!-- Content -->
    <div class="card__content">...</div>
    ...
</div>
```

## CSS

```css styles.css
.card {
    display: flex;
    flex-direction: column;
}

.card__cover {
    height: 20rem;
    width: 100%;
}

.card__content {
    /* Take available height */
    flex: 1;
}
```
