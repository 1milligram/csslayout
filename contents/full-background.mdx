## HTML

```html
<div class="full-background">...</div>
```

## CSS

```css
.full-background {
    /* Center the content */
    align-items: center;
    display: flex;
    flex-direction: column;
    justify-content: center;

    /* Take full size */
    height: 100vh;
    width: 100%;

    /* Background */
    background: url('/path/to/background.jpeg') center center / cover no-repeat;
}
```
